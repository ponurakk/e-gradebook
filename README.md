# Online Gradebook
Project docs are hosted at https://ponurakk.github.io/e-gradebook
## How to Run
Based on your system execute use equivalent manager:

### Linux
```bash
# First you need to build the project
./manager.sh build
# Then you can run it
./manager.sh run
```

### Windows
```bash
# First you need to build the project
./manager.bat build
# Then you can run it
./manager.bat run
```
